# FraudGuard AI | Credit Card Fraud Detection

An AI-powered real-time financial protection system built entirely with Python and Streamlit. This application monitors transactions and uses machine learning patterns to detect fraudulent activities instantly.

## 🚀 Key Features
- **Real-time Dashboard**: Monitor system-wide transaction metrics and prevention stats.
- **AI Analysis Engine**: Instant transaction verification using neural network patterns.
- **Advanced Analytics**: Visualize fraud clusters and model performance metrics.
- **Premium UI/UX**: Dark-themed, glassmorphic design for a professional financial experience.

## 🛠️ Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd "credit card fraud detection"
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```

## 📦 Tech Stack
- **Framework**: Streamlit
- **Data Processing**: Pandas, NumPy
- **Visualizations**: Plotly, Streamlit Charts
- **Machine Learning**: Scikit-learn (Integration ready)
